# Supporting research notes

These are working web-research notes captured while compiling the comparison — pricing,
biomarker panels, features, and trust signals gathered from each service's public website,
app-store listings, and third-party reviews. They are raw notes (with confidence levels and
open questions), not polished prose; the cleaned, synthesized findings live in the main report
(`../report.md`, sections S3–S6 and S9–S10). They contain no personal data.
